# web/public

Static assets served at the site root.

## Required: phaze-logo.png

The Techno Commercial Proposal letterhead (and any future letterhead use)
loads `/phaze-logo.png` — the Phaze Dynamics logo+wordmark lockup on a
transparent/white background (dark wordmark, suitable for a white page).

Drop the file here as `phaze-logo.png`. Until it's present the letterhead
falls back to the plain "Phaze Dynamics" wordmark text (see COMPANY.logoPath
in app/lib/theme.ts).
